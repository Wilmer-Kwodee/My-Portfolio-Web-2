kalo mo push dari 0 pas bener bener kosong
1. git commit -m "first commit"
2. git branch -M main
3. git remote add origin "https://github.com/Wilmer-Kwodee/My-Portfolio-Web-2.git"
4. git push -u origin main

kalo mo update setelah ngubah sesuatu:
1. git status
2. git add .
3. git commit -m "Changes"
4. git push -u origin main
